# Code minimaliste AlpineJS/Tailwind/ParcelJS

Installer les dépendances :

```shell
npm i
```

Lancer le serveur de développement :

```shell
npx parcel src/index.html
```

Ou

```
npm run start
```

Version de production :

```shell
npx parcel build src/index.html
```

Ou

```
npm run build
```
